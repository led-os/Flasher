#Flasher分布式缓存框架
---

 Flasher基于redis3集群服务器，java客户端基于jedis
   
## Flasher特点
* 1、基于Jedis Cluster开发的客户端支持Redis Cluster集群。
* 2、对被调用方（客户端）侵入极少，上手极快。
* 3、支持动态增加节点，客户端自动感知。（zk）
* 4、支持客户端验证与拦截。 token
* 5、异步监控调用数据，支持异步上报。
* 6、方便管理有效的区分业务系统。会员（memmber） 商品（goods）
* 7、支持Falcon协议. 监控系统
* 8、国内一线互联网公司上线项目
